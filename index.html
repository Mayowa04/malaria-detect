Research Context

About This System

Project Overview

Final Year BTech Project by Adegboyegun Oluwamayowa Joseph (SEN/20/5073). An ensemble ML system for early malaria detection in South West Nigeria using clinical patient data.

  • 557 patient records — Ogun & Osun States
  • 15 binary clinical symptom features
  • SMOTE oversampling for class balance
  • Stacking ensemble with LR meta-learner

System Architecture

Full-stack system: HTML/CSS/JS frontend on Vercel communicates with a FastAPI Python backend on Render. Predictions are generated in real time by the trained sklearn ensemble.

  • Frontend: HTML · CSS · Vanilla JS (Vercel)
  • Backend: Python FastAPI (Render.com)
  • ML: scikit-learn · XGBoost · joblib
  • API: RESTful JSON over HTTPS

Dataset Sources

Primary clinical data from confirmed malaria patient records across two South West Nigerian health facilities with RDT and microscopy-confirmed diagnoses.

  • 343 malaria positive cases (61.6%)
  • 214 malaria negative cases (38.4%)
  • Fed. Polytechnic Ilaro Medical Centre
  • PHC Centres, Osogbo, Osun State

⚠️ Disclaimer

This is a research prototype for academic purposes only. It must not replace professional clinical diagnosis, laboratory testing, or clinician judgement under any circumstances.

  • Always confirm with RDT or microscopy
  • Consult a qualified clinician
  • Tool supports — not replaces — diagnosis
  • Not validated for live clinical deployment
