Research Context

About This System

Project Overview

Final Year Project by Adegboyegun Oluwamayowa Joseph (SEN/20/5073). Develops an ensemble-based machine learning model for early malaria detection in South West Nigeria using clinical patient data.

  • 557 patient records from Ogun & Osun States
  • 15 clinical symptom features
  • SMOTE oversampling for class balance
  • Stacking ensemble meta-learner

Methodology

A multi-stage pipeline: data preprocessing → feature extraction → SMOTE balancing → base classifier training → stacking ensemble with Logistic Regression meta-learner → evaluation.

  • Random Forest (300 trees)
  • XGBoost (200 rounds, η=0.05)
  • SVM with RBF kernel
  • LR Meta-Learner (stacking)

Dataset

Primary clinical data collected across South West Nigeria health facilities. Covers patients presenting with malaria-suggestive symptoms, with confirmed diagnoses via RDT and microscopy.

  • 343 positive malaria cases (61.6%)
  • 214 negative cases (38.4%)
  • Federal Polytechnic Ilaro Medical Centre
  • PHC Centres, Osogbo, Osun State

⚠️ Clinical Disclaimer

This system is a research prototype for academic purposes only. It is not validated for clinical use and should not replace professional medical diagnosis, laboratory testing, or clinician judgement.

  • Always confirm with RDT or microscopy
  • Consult a qualified clinician
  • This tool supports, not replaces, diagnosis
  • Not for deployment in live clinical settings
